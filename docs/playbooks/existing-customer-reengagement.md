## Tier 3: Faire-relayed customers (separate B2B touch)

## Capture asks (every email above)

End each personal email with:
1. Honest feedback request (not a public review yet)
2. If they reply positively → ask for permission to include them in a future newsletter quote / share on the site
3. If they had a problem → make it right. Free replacement, no questions

## Tracking

- Use UTM `utm_source=email&utm_medium=reactivation` on every link
- Discount code redemptions = direct attribution
- File replies in HubSpot under the customer's contact

## What NOT to do today

- ❌ Mass-blast all 10 from a generic ESP — looks like spam
- ❌ Send a discount code without context (cheapens the brand)
- ❌ Ask for public reviews before hearing private feedback
- ❌ Pressure for a reorder if they say "not yet"
