# Gateway Communities Channel — Strategic Doctrine

**Status:** LOCKED 2026-04-29 PM (per Ben strategic call)
**Purpose:** Define gateway communities as a permanent, scalable channel for USA Gummies B2B retail growth. Every NPS / state-park / heritage destination has a "gateway town" at the entrance — these towns concentrate tourism-driven gift retail in a way that fits our Made-in-USA + dye-free + impulse-SKU profile better than almost any other channel.

This contract is the canonical reference for gateway-community strategy, prioritization, methodology, and trip planning.

## Why this channel wins

**Pattern fit (matches the customer profile locked in `/contracts/sales-tour-may-2026-prospect-list.md`):**
- High-traffic family + tourist visitor mix
- Premium-positioned shopper paying $4.99–5.99 retail for "Made in USA" story
- Patriotic / heritage / regional pride brand-story buyer
- Register-side strip-clip impulse SKU = perfect format for gift-shop checkout
- Made-in-USA + dye-free + clean label = differentiated from generic candy on shelf

**Channel economics:**
- Per-town concentration: 5-30 independent gift shops in a 1-2 mile walking radius
- Trip economics: founder-driven freight makes per-door cost approach $0 once Ben is in town
- Conversion ramp: getting product into 1 shop usually opens 2-3 more in the same town via word-of-mouth
- Recurring: gift shops reorder seasonally (Memorial Day, July 4 / America 250, Labor Day, Christmas) = 4 reorder windows per year

**Sales assets already exist:** Sales sheets, COA, allergen statement, distribution matrix all built. This is execution at scale, not asset creation.

## Methodology — how to research gateway towns at scale

Apply [`/contracts/research-methodology.md`](research-methodology.md) (Viktor's company-first playbook) to every town. Per-town research sequence:

### Step 1 — Town-level recon (10-15 min)
- Pull town's main street / retail district map (Google Maps interior photos work)
- Identify the 5-15 primary gift shops + hotel gift shops + visitor center retail
- Tag by category: independent gift shop, hotel-attached, NPS concessioner-attached, themed-attraction-adjacent
- Note any iconic / anchor stores (e.g., Eagle's Store in West Yellowstone, Buffalo Bill Center in Cody)

### Step 2 — Per-shop research (per Viktor methodology)
- Company-first recon → org structure, gift shop confirmed, category fit
- Direct website + LinkedIn for named retail buyer / store manager
- Email pattern triangulation
- Confidence % rubric — ≥75% confidence = personalized cold email; below = phone/walk-in only

### Step 3 — Trip integration
- Map each shop to a trip routing window
- Cluster outreach by town: send all shops in same town within 24-48 hours of each other so the "Ben is in town" reference lands consistently
- Walk-in fallback: independent gift shops without web presence (Oatman, Jerome pattern) get walk-in only — no cold email

## Outreach pacing rules

To protect sender reputation while scaling this channel:

- **Per-town cadence:** Send all named-buyer prospects in a single town within 2-3 day window (concentrated, then move on)
- **Daily volume cap:** Maximum 30 personalized cold sends per day from `ben@usagummies.com` (current daily volume is around the right ceiling — 48 today was the upper limit before sender-rep concern)
- **Bounce rule:** Any email pattern below 80% Viktor-confidence → call task only, never cold email. Sender rep degrades fast on bounces
- **Cluster waves:** Hit one gateway town fully (5-15 shops), wait 48 hours for replies, then move to the next town
- **No batch blasts to generic info@ inboxes** — same hard rule as `/contracts/sales-tour-may-2026-prospect-list.md`
