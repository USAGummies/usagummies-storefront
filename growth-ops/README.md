# Growth Ops: Outside OpenClaw Nuclear Stack

This folder is a standalone execution stack to drive organic reach and traffic without touching website code.

## What this build generated
1. Authority profile takeover copy pack
2. 30-day parasite SEO syndication calendar (2 posts/day)
3. Influencer + affiliate discovery pipeline (search-ready lead rows)
4. PR + citation distribution queue with direct submission links
5. UTM registry and daily scoreboarding template for kill/scale decisions

## Regenerate
`node scripts/growth-ops/build-growth-ops-assets.mjs`

## Operating rule
Only double down on channels that produce sessions + orders in the scoreboard. De-prioritize channels with high effort and no conversion within 7 days.
