# USA Gummies Review Insert Card

Physical insert card to be included in every bag/order. Drives customers to leave an Amazon review and spin the wheel for a discount.

---

## Card Specifications

- **Size**: 3.5" x 2" (standard business card size)
- **Stock**: 16pt matte or glossy cardstock
- **Corners**: Rounded (1/8" radius)
- **Printing**: Full color, double-sided (4/4)
- **Finish**: Soft-touch matte recommended for premium feel (glossy for budget)

---

## FRONT OF CARD

### Layout
Centered layout, clean and inviting. Background: cream/parchment (#f8f5ef) to match brand.

### Copy

```
[USA Gummies Logo - top center, small]

---------------------------------------

      YOU'RE A VIP  [US flag emoji]

        Scan for a
       FREE REWARD

    [QR CODE - large, centered]
    usagummies.com/review

  Share your experience,
   spin the wheel, win!

---------------------------------------
```

### Design Notes (Front)
- Logo: Small, ~0.75" wide at the top
- "YOU'RE A VIP" in Oswald Bold, navy (#1B2A4A), 14pt
- "Scan for a FREE REWARD" in Oswald Bold, red (#c7362c), 11pt
- QR Code: ~1.2" x 1.2", centered, links to https://www.usagummies.com/review
- QR Code should have USA Gummies logo or flag embedded in center if possible
- URL below QR: Space Grotesk, 7pt, gray (#5f5b56)
- Bottom tagline: Space Grotesk, 8pt, navy (#1B2A4A)
- Border: thin red (#c7362c) line around card edge, 2pt inside margin

## QR Code Details

- **URL**: `https://www.usagummies.com/review`
- **UTM tracking**: Consider `https://www.usagummies.com/review?utm_source=insert_card&utm_medium=print&utm_campaign=review_velocity`
- **Error correction**: High (H) to allow logo overlay
- **Foreground color**: Navy (#1B2A4A)
- **Background**: White or cream (#f8f5ef)
- **Size on card**: 1.2" x 1.2" minimum
- **Generate at**: qr-code-generator.com or use a library

## Print Production Notes

### Recommended Print Services

1. **Vistaprint** (vistaprint.com)
   - Standard business cards, 250 for ~$15-25
   - Upload-ready template support
   - 5-7 business day delivery

2. **MOO** (moo.com)
   - Premium quality, rounded corners included
   - 50 for ~$20, 400 for ~$60
   - Luxe paper options available

3. **GotPrint** (gotprint.com)
   - Budget option, 500 for ~$15
   - Rounded corners extra ~$5
   - Fast turnaround (3-5 days)

4. **4over** (4over.com)
   - Trade printer, best for bulk (1000+)
   - 1000 for ~$25-40
   - Requires print-ready files

### Cost Estimates

| Quantity | Budget      | Premium     |
|----------|-------------|-------------|
| 250      | $15-20      | $30-40      |
| 500      | $20-30      | $50-70      |
| 1,000    | $25-40      | $70-100     |
| 5,000    | $60-100     | $200-300    |

### File Preparation
- Deliver as PDF with 0.125" bleed on all sides
- CMYK color mode (not RGB)
- 300 DPI minimum
- Include crop marks
- Convert fonts to outlines

## Insertion Instructions

- Include one card per bag (place inside the resealable bag)
- For 5-pack bundles: include one card in the outer shipping box (not in each bag)
- Card should be visible when customer opens the package
- Consider placing QR code face-up for immediate visibility
